# Translate face names using stored flip information

Translate face names using stored flip information

Translate face names to account for scale direction flips

## Usage

``` r
translate_face_names_from_flips(face_names, scale_flips)

translate_face_names_from_flips(face_names, scale_flips)
```

## Arguments

- face_names:

  Character vector of face names

- faces:

  Character vector of face names

- flips:

  List with x, y, z flip indicators

- scale_x, scale_y, scale_z:

  Scale objects for each axis

## Value

Character vector of translated face names

Character vector of translated face names

# Get axis name for a single scale with proper fallback hierarchy

Get axis name for a single scale with proper fallback hierarchy

## Usage

``` r
get_scale_names(scale_obj, axis_name)
```

## Arguments

- scale_obj:

  Scale object (can be NULL for z-axis)

- axis_name:

  Axis name ("x", "y", or "z")

## Value

Single axis name string

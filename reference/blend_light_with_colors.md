# Apply shading to colors based on lighting values

Apply shading to colors based on lighting values

## Usage

``` r
blend_light_with_colors(colors, light_values, lighting)
```

## Arguments

- colors:

  Vector of color values (hex or named colors)

- light_values:

  Numeric vector of lighting values

- lighting:

  Lighting specification with shade_mode and shade_strength

## Value

Vector of modified colors
